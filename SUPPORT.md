# Support

## Usage questions

- Open a GitHub issue with:
  - runtime (`node` / `deno` / `bun` / `web`)
  - bytefold version
  - minimal reproducible example

## Bug reports

- Use GitHub issues with expected vs actual behavior and sample archive/input.

## Security reports

- Follow `SECURITY.md` and use private disclosure channels.
