# Code of Conduct

This project follows a respectful, harassment-free collaboration model for all participants.

## Expected behavior

- Use inclusive language.
- Focus reviews and discussions on technical content.
- Assume good intent and provide constructive feedback.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory or abusive language.
- Publishing private information without consent.

## Reporting

- Report conduct issues privately to `ismail.elkorchi@gmail.com`.
- Security issues should be reported via `SECURITY.md` channels, not this process.
